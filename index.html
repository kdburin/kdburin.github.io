Statistical Inference & Distribution Analysis
Five-week progressive statistical analysis comparing distributions A, B, C, and D using histograms, ROC curves, two-sample t-tests, one-way ANOVA, and assumption diagnostics. Showed that visual EDA (histogram shape, spread, center) reliably predicts formal test outcomes — A vs C achieved AUC 0.757 and p < 2.2e-16, while D's extreme variance (σ² ≈ 1,253) violated normality and homoscedasticity assumptions. Validated conclusions using Shapiro-Wilk, Levene's test, Q-Q plots, and Tukey HSD post-hoc comparisons.
